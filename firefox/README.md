# Source of Firefox add-on of WikiRank
Assessing quality and popularity of the Wikipedia article from current Firefox tab

Working add-on can be found at: https://addons.mozilla.org/en-US/firefox/addon/wikirank/

Short video on how it works: https://twitter.com/wikiranknet/status/1322636459665403906

To test how it works you need to open any Wikipedia article in Firefox and then press WikiRank icon, for example: https://en.wikipedia.org/wiki/United_States
